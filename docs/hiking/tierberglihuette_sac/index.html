Tierberglihütte SAC

July 23, 2023 by Pat and Bruno  ‐  1 min read

Difficulty T4 & K4
⇄ Distance 5.45 km
↑ Up 670 m
↓ Down 670 m
Notable Locations Umpol, Tierbergli Via Ferrata, Tierberglihütte SAC
Public Transit Access? No

Overview Map Gallery

Quick Summary #

  • A via ferrata and alpine hiking path near the Sustenpass.
    • There was no public transit available to get to the starting location.
    • We saw one group that hired an alp taxi to pick them up at the Umpol parking lot, so there may be an option to charter a ride from one of the nearby villages.
  • We only ended up doing the first half of the via ferrata and then decided to continue to the hut via the alpine hiking route.
    • There were a handful of slightly overhanging sections on the first part of the via ferrata so our arms were already a little fatigued by the point where we decided to make our detour.
    • The via ferrata crossed the hiking path at one location, which is where we decided to make this detour.
  • We were here on a weekend with excellent weather so the route was quite busy.
