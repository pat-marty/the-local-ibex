Val Mulix

October 28, 2023 by Pat and Bruno  ‐  1 min read

Difficulty T3
⇄ Distance 12.4 km
↑ Up 830 m
↓ Down 830 m
Notable Locations Preda, Naz, Val Mulix
Public Transit Access? Yes

Overview Map Gallery

Quick Summary #

  • Autumn hike in Graubünden near Preda.
  • Parked the car near Naz.
    • There is a nearby train station in Preda which could serve as an alternative.
  • Hiked up along the Val Mulix and continued south towards the Alp Mulix.
    • From this point onwards, there was only really one hiking path for us to follow.
  • The path was quite icy so we were glad to have our trekking poles and snow spikes (lightweight crampons) with us.
  • We did not go all the way up to the Lai Negr because the icy conditions on some of the boulder fields were a little precarious.
  • This was a very quiet route — we didn’t see anyone else the entire day.
  • There were many larch trees along the way which added a beautiful splash of colour to the scenery.
